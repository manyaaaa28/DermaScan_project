## 🩺 DermaScan: AI-Powered Skin Diagnosis

DermaScan is a Flask-based web application designed to assist in the early detection of skin conditions. By leveraging a Convolutional Neural Network (CNN), the system analyzes uploaded images of skin lesions to provide real-time diagnostic predictions and personalized skincare recommendations.

https://dermascan-6te4.onrender.com/

## ✨ Features

AI Diagnosis: Classifies skin conditions such as Acne, Rashes, and Allergies using deep learning.

Dynamic Dashboard: Tracks your skin's "Clarity Score" and maintains a history of previous scans.

Smart Recommendations: Provides instant, condition-specific advice based on AI findings.

Interactive Chatbot: Integrated "DermaGPT" to answer general dermatology questions.

Appointment Booking: Easy access to both online and offline medical consultations.

## 🛠️ Tech Stack

Frontend: HTML5, CSS3 (Modern Purple Neon Theme), JavaScript.

Backend: Python, Flask.

Machine Learning: TensorFlow, Keras (CNN Architecture).

## 🧪 Usage
1. Run the backend
2. Upload an image
3. View prediction & recommendations

## ⚠️Deployment Note
Cloud Hosting Status: This project is optimized for local performance.

Due to the high computational requirements and the large file size of the trained TensorFlow/Keras model (.h5), deployment on free-tier cloud platforms (like Render or Vercel) is restricted. These platforms often have strict RAM limits (e.g., 512MB) that are exceeded when loading heavy deep learning models for real-time inference. For a full production experience, a dedicated server with higher memory capacity or GPU support is recommended.
